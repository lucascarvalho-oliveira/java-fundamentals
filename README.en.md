# Java Fundamentals

This repository contains **exercises, code examples, and small projects developed during my Java studies**.
The purpose of this repository is to practice the **fundamentals of the Java language**, understand essential programming concepts, and gradually progress toward more complex software development.
The code here represents continuous practice with **programming logic, core language structures, and Object-Oriented Programming (OOP)**.

## Repository Purpose

This repository was created to:

* Practice **Java language fundamentals**
* Develop **programming logic**
* Apply **Object-Oriented Programming (OOP) concepts**
* Experiment with different language features
* Track my progress while learning Java

## Topics Covered

Some of the main topics practiced in this repository include:

* Variables and data types
* Operators
* Conditional structures (`if`, `switch`)
* Loop structures (`for`, `while`)
* Methods and functions
* Arrays and collections
* Classes and objects
* Encapsulation
* Inheritance
* Package organization
