# Ranking de Notas em Java
Este programa em Java calcula informações de um conjunto de notas dos alunos.<br>
Ele realiza três tarefas principais:
- Calcula a média das notas
- Conta quantos alunos ficaram acima da média
- Identifica a maior nota

## Tecnologias Utilizadas
- Java
- Estruturas básicas: 'array', 'for', 'if'

## Como o programa funciona
1. Um vetor 'double[]' armazena as notas dos alunos.
2. Um laço 'for' soma todas as notas.
3. A média é calculada dividindo a soma pelo total de notas.
4. Outro laço conta quantas notas são maiores que a média.
5. Um último laço identifica a maior nota do vetor.
6. Os resultados são exibidos no console.
