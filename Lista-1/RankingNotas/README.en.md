# Grade Ranking in Java
This Java program calculates information from a set of students' grades.
It performs three main tasks:

* Calculates the average grade
* Counts how many students scored above the average
* Identifies the highest grade

## Technologies Used

*Java
*Basic structures: `array`, `for`, `if`

## How the Program Works

1. A `double[]` array stores the students' grades.
2. A `for` loop sums all the grades.
3. The average is calculated by dividing the total sum by the number of grades.
4. Another loop counts how many grades are higher than the average.
5. A final loop identifies the highest grade in the array.
6. The results are displayed in the console.
